** -- Info Commands --
**   HELP                 : Voir l'aide
**   XDLFULL              : Liste tous les paquets proposés
**   XDLGROUP [groupe]    : Voir les paquets de <groupe>
**   XDL                  : Liste les fichiers par groupes et les fichiers sans groupe
**   XDLOCK               : Voir tous les paquets protégés
**   XDTRIGGER            : Voir tous les paquets avec des déclencheurs dynamiques
**   FIND critère         : Liste des paquets qui correspond à <critère>
**   XDS                  : Sauvegarde du fichier statistiques
**   DCL                  : Liste les transferts en cours
**   DCLD                 : Liste détaillée des transferts en cours
**   TRINFO identifiant   : Liste les informations du transfert <identifiant>
**   GETL                 : Liste des téléversement en attente
**   QUL                  : Liste la file d'attente
**   IGNL                 : Voir la liste des utilisateurs ignorés
**   LISTUL [dossier]     : Voir le contenu du répertoire des fichiers reçus
**   DISKFREE             : Voir l'espace libre du répertoire des fichiers reçus
**   CHANL [net]          : Voir la liste des salons et des utilisateurs du <net>
** -- Transfer Commands --
**   CLOSE [id]           : Coupe le transfert <identifiant>
**   CLOSEU [id]          : Coupe l'envoi <identifiant>
**   ACCEPTU min [masque] : Accepte les fichiers de <masque> pendant <x> min
**   GET serveur pseudonyme n [mot_de_passe] : Demander le paquet <n> du robot <pseudonyme>
**   CLOSEGET serveur pseudonyme : Annule la demande de paquet pour le robot <pseudonyme>
**   RMQ [position]       : Supprime l'entrée <position> de la file d'attente
**   RMIQ [position]      : Supprime l'entrée <position> de la file d'attente d'inactivité
**   RMALLQ               : Enlèves toutes les entrées de la file d'attente d'inactivité et de la file d'attente normale
**   NOMIN identifiant    : Désactive la vitesse minimum du transfert <identifiant>
**   NOMAX identifiant    : Désactive la vitesse maximum du transfert <identifiant>
**   UNLIMITED identifiant : Désactive les limites de bande passante pour le transfert <identifiant>
**   MAXSPEED identifiant x : Définit la limite maximale de bande passante à <x> Kio/s pour le transfert transfert <id>
**   SEND pseudonyme n [net] : Envoi le paquet <n> à <pseudonyme>
**   QUEUE pseudonyme n [net] : Met le paquet <n> pour <pseudonyme> en file d'attente
**   IQUEUE pseudonyme n [net] : Met le paquet <n> pour <pseudonyme> en file d'attente d'inactivité
**   BATCH pseudonyme n-m [serveur] : Met les paquets <n> à <m> en file d'attente d'inactivité pour <pseudonyme>
**   PSEND salon style [net] : Envoi le contenu de « XDCC LIST » avec le style <(full|minimal|summary)> (ie : complet|minimal|résumé) sur le <salon>
**   QSEND [id]           : Démarrer le transfert d'une file d'attente
**   IQSEND [id]          : Passer un envoi de la file d'attente d'inactivité à la file d'attente normale
**   SLOTSMAX [x]         : Changement temporaire du paramètre « slotsmax » à <x> connexions
**   QUEUESIZE [x]        : Changement temporaire du paramètre « queuesize » à <x> éléments
**   REQUEUE x y          : Change la position de la file d'attente de <x> à <y>
**   REIQUEUE x y         : Change l'entrée de la file d'attente d'inactivité de la position <x> à <y>
** -- Pack Commands --
**   INFO n               : Voir les informations du paquet <n>
**   REMOVE n [m]         : Enlève le paquet <n> ou les paquets de <n> à <m>
**   REMOVEDIR dossier    : Enlève tous les paquets situés dans <dossier>
**   REMOVEGROUP groupe   : Enlève tous les paquets situés dans <groupe>
**   REMOVEMATCH critère  : Enlève tous les paquets contenant ce terme
**   REMOVELOST [critère] : Supprimer les paquets orphelins correspondants à ce critère
**   RENUMBER x [y] z     : Déplace les paquets de <x> à <y> à la position <z>
**   SORT [champ] [champ] : Trie les paquets par <champ>. Options possibles : « NAME », « DESC », « GROUP », « PATH », « SIZE », « TIME »,  « ADDED » (nom, description, groupe, chemin d'accès, taille du fichier, date de modification, date d'ajout)
**   ADD fichier          : Ajoute le fichier <fichier>
**   ADDDIR dossier       : Ajoute les fichiers de <dossier>
**   ADDNEW dossier       : Ajoute les nouveaux fichiers de <dossier>
**   ADDGROUP groupe dossier : Ajoute les nouveaux fichiers de <dossier> dans <groupe>
**   ADDMATCH critère     : Ajoute les nouveaux fichiers correspondant à <modèle>
**   AUTOADD              : Explore « autoadd_dir » pour rechercher de nouveaux fichiers
**   AUTOCANCEL           : Annule les actions d'ajouts et de suppressions en attente
**   AUTOGROUP            : Créé un groupe pour chaque répertoire avec des paquets
**   NOAUTOADD x          : Désactive AUTOADD pendant les <x> prochaines minutes
**   CHFILE n nouveau_nom : Change le nom du paquet <n> en <nouveau_nom>
**   NEWDIR dossier nouveau_dossier : Renomme le répertoire de tous les paquets correspondant
**   CHDESC n [texte]     : Change la description du paquet <n> en <texte>
**   CHNOTE n [texte]     : Change la note du paquet <n> en <texte>
**   CHTIME n [texte]     : Change la date d'ajout du paquet <n> en <texte>
**   CHMINS n [m] x       : Change la vitesse minimale des paquets <n> à <m> en <x> Kio/s
**   CHMAXS n [m] x       : Change la vitesse maximale des paquets <n> à <m> en <x> Kio/s
**   CHLIMIT n [m] x      : Change la limite de téléchargements des paquets <n> à <m> à <x> transferts par jour
**   CHLIMITINFO n [texte] : Change l'annonce de la vitesse limite de téléchargement du paquet <n> en <texte>
**   CHTRIGGER n [texte]  : Change le déclencheur du paquet <n> en <texte>
**   DELTRIGGER n [m]     : Efface le déclencheur des paquets <n> à <m>
**   CHGETS n [m] x       : Change le nombre de téléchargements des paquets <n> à <m> à <x>
**   CHCOLOR n [m] x[,b][,s] : Défini les paquets <n> à <m> avec la couleur <x>, avec un fond <b> et un style <s>
**   LOCK n [m] mot_de_passe : Protège les paquets <n> à <m> avec <mot_de_passe>
**   UNLOCK n [m]         : Retire la protection des paquets <n> à <m>
**   LOCKGROUP groupe mot_de_passe : Protège tous les paquets de <groupe> avec <mot_de_passe>
**   UNLOCKGROUP groupe   : Retire la protection tous les paquets de <groupe>
**   RELOCK ancien_mot_de_passe mot_de_passe : Bloque tous les paquets avec <ancien_mot_de_passe> par <mot_de_passe>
**   GROUPDESC group msg  : Change la description de <groupe> en <texte>
**   GROUP n groupe       : Change le groupe du paquet <n> en <groupe>
**   MOVEGROUP n m group  : Déplace le groupe vers <groupe> pour les paquets <n> à <m>
**   REGROUP groupe nouveau : Change tous les paquets de <groupe> en groupe <nouveau>
**   NEWGROUP groupe dossier : Change tous les paquets de <dossier> vers <groupe>
**   ANNOUNCE n [texte]   : Annonce <texte> pour le paquet <n> dans tous les salons
**   NEWANN n [salon] [serveur] : Annonce les derniers <n> paquets dans tous les salons
**   MANNONCE n m [texte] : Annonce <texte> pour <n> à <m> dans tous les salons
**   CANNOUNCE salon n [texte] : Annonce <texte> pour le paquet <n> dans <salon>
**   SANNOUNCE n [m]      : Annonce courte pour les paquets <n> à <m> dans tous les salons
**   NOANNOUNCE x         : Désactive toutes les annonces pendant les <x> prochaines minutes
**   ADDANN fichier       : Rajoute et annonce un nouveau paquet
**   MD5 [n] [m]          : Calcule MD5 et CRC32 des paquets <n> à <m>
**   CRC [n] [m]          : Vérifie le CRC32 des paquets <n> à <m>
** -- File Commands --
**   FILEMOVE fichier nouveau_nom : Renomme le fichier sur le disque dur à partir de son ancien nom
**   MOVEFILE n nouveau_nom : Renomme sur le disque dur le fichier <n> en <nouveau_nom>
**   MOVEGROUPDIR groupe dossier : Déplace tous les fichiers de <groupe> vers <dossier>
**   FILEDEL fichier      : Efface le fichier sur le disque dur
**   FILEREMOVE n [m]     : Enlève le paquet <n> ou de <n> à <m> et l'efface du disque dur
**   SHOWDIR dossier      : Liste le contenu de <dossier> sur le disque dur
**   MAKEDIR dossier      : Créer un nouveau répertoire sur le disque dur
**   FETCH fichier URL    : téléchargement l'<URL> et sauvegarde le <fichier> dans le répertoire d'envoi
**   FETCHCANCEL [id]     : Arrête le téléchargement du fichier avec l'<identifiant>
** -- Misc Commands --
**   MSG pseudonyme message : Envoi <message> à l'utilisateur ou sur le salon <pseudonyme>
**   AMSG texte           : Annonce <texte> sur tous les salons
**   MSGNET réseau pseudonyme message : Envoi <message> à l'utilisateur ou sur le salon <pseudonyme>
**   MESG message         : Envoi <message> à tous les utilisateurs en cours de transfert
**   MESQ message         : Envoi <message> à tous les utilisateurs en file d'attente
**   IGNORE x masque      : Ignore <masque> (pseudonyme!identite@hote) pour <x> minutes, (caractères joker autorisés)
**   UNIGNORE masque      : N'ignore plus le <masque>
**   BANNHOST x masque    : Stoppe les transferts et ignore <masque> (pseudonyme!identite@hote) pour <x> min
**   BANNNICK pseudonyme [serveur] : Stoppe les transferts et enlève <pseudonyme> de la file d'attente
**   NOSAVE x             : Désactive la sauvegarde automatique pour les <x> prochaines minutes
**   NOSEND x [texte]     : Désactive « XDCC SEND » pour les <x> prochaines minutes
**   NOLIST x             : Désactive « XDCC LIST » et « PLIST » pour les <x> prochaines minutes
**   NOMD5 x              : Désactive le calcul MD5 et CRC32 pendant les <x> prochaines minutes
**   MSGREAD              : Voir le journal
**   MSGDEL               : Efface le journal
**   RMUL fichier         : Efface un fichier dans le répertoire d'envoi
**   RAW commande         : Envoi <commande> aux serveurs
**   RAWNET serveur commande : Envoi <commande> aux serveurs
**   LAG [net]            : Voir le temps de latence des réseaux
** -- Bot Commands --
**   SERVERS [net]        : Voir la liste des serveurs
**   HOP [salon] [serveur] : Part et rejoint un salon
**   NOCHANNEL x [salon]  : Quitter le chan pour <x> minutes
**   JOIN salon [serveur] [clé] : Joint le salon jusqu'au prochain rechargement de la configuration
**   PART salon [serveur] : Quitte le salon jusqu'au prochain rechargement de la configuration
**   JUMP serveur [net]   : Bascule vers un autre serveur ou vers <serveur>
**   SERVQC [net]         : Efface la file d'attente du serveur
**   STATUS               : Voir les informations utiles
**   REHASH               : Relire le fichier de configuration et le prendre en compte
**   BOTINFO              : Voir les informations sur le statut du robot
**   MEMSTAT              : Voir les informations sur l'usage de la mémoire
**   VERSION              : Show Information about iroffer version
**   CLEARRECORDS         : Efface les transferts, la bande passante, le temps de fonctionnement et les limites de transfert
**   CLEARGETS            : Efface les compteurs de téléchargement pour chaque paquet, le temps de fonctionnement et le total envoyé
**   REDRAW               : Redessine l'écran
**   DELHIST              : Efface l'historique de la console
**   QUIT                 : Ferme ce DCC CHAT
**   EXIT                 : Ferme ce DCC CHAT
**   LOGOUT               : Ferme ce DCC CHAT
**   CHATME               : Vous envoie une requête DCC CHAT
**   CHATL                : Liste les informations DCC CHAT
**   CLOSEC [id]          : Ferme le DCC CHAT <identifiant>
**   DEBUG x              : Fixe le niveau de verbosité à <x>
**   CONFIG Valeur de la clé : Définir la variable de configuration <clé> à <valeur>
**   PRINT clé            : Affiche la variable de configuration <clé>
**   IDENTIFY [net]       : Renvoye le mot de passe à NickServ
**   HOLDQUEUE [x]        : Bloque les envois depuis la file d'attente (« 1 ») ou retire le blocage (« 1 »)
**   OFFLINE [net]        : Fermer le réseau indiqué ou tous les réseaux
**   ONLINE [net]         : Se connecter au réseau indiqué ou à tous les réseaux
**   RUBY méthode [paramètres] : Appel une méthode dans le « ruby_script » avec les paramètres <paramètres>
**   SHUTDOWN delai       : Fermeture d'Iroffer, <delai> est « now » (maintenant), « delayed » (retardé, une fois les envois terminés), ou « cancel » (annulé)
**   BACKGROUND           : Basculer en arrière-plan
**   DUMP                 : Ecrire un cliché de la configuration dans le journal
**   RESTART              : Arrêter et redémarrer le robot
**   CRASH                : Ecrire un cliché de la configuration dans le journal et quitter
